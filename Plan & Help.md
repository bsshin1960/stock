# KODEX 200 AI 주식 예측 프로그램 사용 매뉴얼 (Plan & Help)

본 프로그램은 매 영업일 오후 3시 장 마감 시점의 국내외 금융 지표 및 시장 뉴스를 분석하여 다음 날 **KODEX 200 ETF의 시초가 등락을 예측**하는 인공지능 기반 주식 분석 솔루션입니다. 글로벌 4대 거대언어모델(LLM)인 Gemini, ChatGPT, Claude, Grok의 독립적인 분석 관점과 실시간 선물/매크로 지표, 실시간 속보 뉴스 및 증권가 소문을 종합 결합하는 **4영역 다차원 가중치 합산 예측 엔진**을 탑재하였습니다.

---

## 1. 주요 특징 및 핵심 기능
1. **정량/정성 데이터 통합 수집**:
   - KODEX 200 시세 데이터 및 기술적 지표(이동평균, RSI, MACD, 볼린저 밴드) 연산.
   - **삼성전자 및 SK하이닉스** 등 KODEX 200지수에 지대한 영향력을 가진 대형 기술주의 당일 주가 흐름 반영.
   - **미국 야간 선물(나스닥 선물) 및 국내 자산(Kodex200)**, **한국 선물(코스피 200 선물)**의 변동성 데이터 융합.
   - 원/달러 환율 및 **달러/엔 환율(일본 엔화 상황)**, **일본 닛케이 225(Nikkei 225)**의 변동성 추적. (수출 경쟁력 및 아시아 증시 동조성 분석)
   - 글로벌 시장의 심리를 보여주는 **VIX 공포 지수** 수집.
   - 당일 주요 포털 금융 뉴스 실시간 크롤링 및 신뢰성 있는 **증권가 소문/이슈(풍문)** 데이터를 반영.
2. **다차원 종합 예측(Consensus v4.0)**:
   - **4대 핵심 AI 예측 분석 (50%)**: Gemini, ChatGPT, Claude, Grok의 가중 평균 예측치를 50% 비중으로 반영.
   - **글로벌 매크로 실시간 대시보드 (30%)**: 16개 금융 및 기술적 분석 지표를 가중 평균하여 30% 비중으로 반영.
     - **개별 매크로 지표 가중치 (초기값 0% 설정 및 분석 시 동적 변경)**: 코스피 선물, 나스닥 선물, Kodex200, 원/달러 환율, 달러/엔 환율, 일본 닛케이, VIX 공포지수, 미 10년 국채금리, WTI 국제 유가, 국내 금리, 국내 채권금리, 필라델피아 반도체, 달러 인덱스, 금 선물, 기술적분석1, 기술적분석2 지표들의 가중치 비중이 실시간 분석 시점의 변동폭과 시간대에 따라 자동 조절 및 정규화됩니다.
   - **실시간 속보 뉴스 (15%)**: 당일 수집된 뉴스 헤드라인에 대한 정량 감성 긍부정 연산 등락률을 15% 비중으로 반영.
   - **증권가 소문/이슈 (5%)**: 당일 선별된 루머 찌라시 텍스트에 대한 감성 분석 결과를 5% 비중으로 반영.
3. **사용자 친화적 Premium Dark UI & 시인성 개선**:
   - 모던하고 미려한 다크 테마 기반 대시보드.
   - **동적 실시간 시각 및 서브타이틀 고정**: 최상단 헤더에 현재 분석 기준 시각(`YYYY년 MM월 DD일 HH시 MM분 기준`)을 표기하며, **밝은 연두색 굵은 글씨** 스팬을 적용하여 가독성을 높였습니다. 분석 완료 시 시각이 동적으로 업데이트되며 서브타이틀의 굵기(BOLD), 폰트크기, 테마 맞춤 색상이 온전히 유지됩니다.
   - **시총 TOP10 현재 주가 표기**: 우측 "시총 TOP10 회사 주가" 패널에 **현재 주가(예: 72,100원)**를 중간에 추가하여, 직관성을 높였습니다 (`[회사명] [현재 주가] [변동%]`). 테마 변경 시에도 주가와 회사명의 색상이 정상 연동됩니다.
   - **하락 전망 시 밝은 연두색 테두리 박스**: 상승 시 초록색 테두리, 하락 시 눈이 편안하고 가시성이 뛰어난 밝은 연두색 테두리 박스(`#AEEA00`)와 연두색 텍스트 지표로 시각화합니다.
   - **뉴스 / 소문 좌우 독립 박스 분리**: 기존의 탭 토글 방식에서 벗어나, 가로 574px 및 세로 170px 규격의 두 개의 독립된 박스로 분리 배치하여 뉴스와 소문을 실시간으로 동시에 상시 모니터링할 수 있도록 개편하였습니다.
   - **타이틀 스타일 통일 및 상징 아이콘 배치**: 대시보드의 주요 박스 및 세션 제목의 색상과 크기를 **밝은 연두색 (`#AEEA00`)**, **`size=13`**, **BOLD** 속성으로 일관되게 통일하고, 각 섹션을 상징하는 아이콘(ANALYTICS, FLAG, PSYCHOLOGY, PUBLIC, ARTICLE, RECORD_VOICE_OVER, MONITOR)을 추가하여 직관성을 극대화하였습니다.
   - **스크롤 및 레이아웃 최적화**: 메뉴바 아래 밀착 레이아웃 적용, 수직 가상 스크롤바 드래그 감도(1/5 감도 제한) 최적화로 빠른 이동 제공, 최하단 차트 탑재를 고려한 스크롤 연산 최적화(`content_height = 1330.0`), 차트 박스 하단 여유 마진(10mm) 적용.
   - **시총 TOP10 주가/등락률 한 줄 표기**: 시총 TOP10 패널에서 회사명·주가·등락률을 하나의 행에 인라인으로 표시하며, 주가는 왼쪽, 등락률은 오른쪽으로 정렬(`SPACE_BETWEEN`)하여 비교가 용이하도록 개선하였습니다.
   - **최종 예측 박스 레이아웃 정렬 개선 (2026-06-17)**:
     - 상승/하락 전망 텍스트는 **왼쪽 정렬**, 예상 등락률(%)은 **오른쪽 정렬**로 변경하여 시각적 구분을 명확히 하였습니다.
     - "오늘 대비 변동" 및 "예상 시가" 서브텍스트를 **굵은 검정색**(`BOLD`, `#000000`)으로 변경하여 가독성을 향상시켰습니다.
   - **AI 분석 카드 레이아웃 개선 (2026-06-17)**:
     - 개별 AI 카드의 예측 주가 텍스트를 **굵은 검정색**(`BOLD`, `#0F172A`)으로 변경하였습니다.
     - 주가 행과 불릿 분석 내용 간의 줄 간격을 **최소화**(`spacing=0`, `margin top=-10`)하였습니다.
     - 불릿 분석 내용의 줄 간격을 **1.5배**(`TextStyle height=1.5`)로 증대하여 내용 가독성을 개선하였습니다.
   - **예측 이력 정렬 및 상승 전망 위치 개선 (2026-06-17)**:
     - 예측 적중률 박스의 이력 목록을 "대기" 항목 우선 정렬로 고정하였습니다.
     - 컬럼 너비를 `[80, 90, 110, 120, 45]`로 명시적으로 고정하여 정렬이 유지되도록 개선하였습니다.
4. **주요 주가 예측 요인 및 기여도 분석 리스트**:
   - **중요도 순 실시간 정렬**: 거시 경제 지표 및 기술적 분석 지표를 분석 시점의 영향력(절대 가중치 * 변동률)을 기준으로 내림차순 정렬하여 보여줍니다.
   - **정량 지표 기여도 한눈에 확인**: 각 지표의 현재 수치, 전일 대비 변동률, 부여된 가중치 및 기여도(%)가 일목요연하게 표시되어 예측의 주요 원인을 직관적으로 파악할 수 있습니다.
   - **마우스 휠 스크롤 지원**: 마우스 휠로 부드럽게 목록을 아래위로 넘기며 24개의 핵심 지표를 확인 가능합니다.
5. **보고서 자동 저장**:
   - 상단 메뉴바의 **[파일] > [보고서 저장]** 또는 단축 메뉴를 통해 매 예측 결과를 누적하여 Markdown 포맷 보고서(`reports/` 폴더 내 저장)로 안전하게 내보낼 수 있습니다.
6. **단독 실행형 EXE 파일 제공**:
   - 루트 폴더에 빌드된 `Kodex200_AI_Stock_Predictor.exe` 파일은 파이썬이 설치되지 않은 환경에서도 단독으로 실행 가능합니다.
   - 실행 시 실행 파일과 같은 경로에 `settings.json`, `history.json`, `reports/` 폴더가 없으면 자동으로 생성하므로 단일 실행 파일만으로도 어디서나 가동이 가능합니다. (기존 환경 복사 시에는 이 설정/이력 파일들을 같이 복사하여 이동하면 됩니다.)

프로젝트는 모듈화된 파이썬 구조로 제작되어 유지보수와 기능 확장이 매우 용이합니다.

```
C:\Temp\Antigrvity\stock\

  ├─ .env                   # API Key 관리 환경 변수 설정 파일 (GUI 키 없을 때의 폴백용)
  ├─ Plan & Help.md         # [본 문서] 사용자 가이드 및 설명서
  └─ src/
       ├─ __init__.py       # 패키지 선언 파일
       ├─ config.py         # 금융 지표 티커 설정 및 매크로/AI 가중치 정의

       ├─ reporter.py       # 예측 보고서 마크다운 변환 및 저장
       └─ app.py            # Flet GUI 컴포넌트 및 비동기 이벤트 핸들러
```

## 3. 프로그램 실행 방법 (Quick Start)

### 3.1. 단독 실행 파일 (EXE) 실행 (가장 추천)
1. 루트 폴더에 있는 **`Kodex200_AI_Stock_Predictor.exe`** 파일을 더블 클릭하여 즉시 실행합니다.
2. 파이썬이나 기타 의존성을 전혀 설치하지 않아도 바로 가동되며, 프로그램 실행 경로에 설정 파일(`settings.json`), 이력 파일(`history.json`), 보고서 폴더(`reports/`)가 자동으로 생성 및 영구 누적 관리됩니다.

### 3.2. 원클릭 실행 (배치 파일)
1. 프로그램 폴더 내에 있는 **`run.bat`** 파일을 더블 클릭하여 실행합니다.
2. 프로그램이 필요한 패키지(`flet`, `yfinance` 등)를 알아서 다운로드 및 설치한 후 파이썬 환경 하에서 예측 GUI 화면을 띄워 실행합니다.

### 3.3. 수동 실행 (터미널)
파워쉘이나 cmd 터미널에서 다음 명령어를 순서대로 실행합니다:
```bash
# 1. 의존성 라이브러리 설치
pip install -r requirements.txt

# 2. 프로그램 가동
python main.py
```

## 4. 실시간 한글 진행 상황 (Process flow)
프로그램이 분석을 실행하면, GUI의 상태표시줄에 모든 분석 진척 상황이 **한글**로 알기 쉽게 표시됩니다:

1. **`대형주, 한일 선물지수, 환율 및 공포지수 크롤링 중...`**
   - KODEX 200, 삼성전자, SK하이닉스 가격 및 해외 선물 지수, 환율, VIX 인덱스 등의 데이터를 수집하는 단계입니다.
2. **`AI 모델 분석 시뮬레이션 및 API 응답 수립 중...`**
   - 4대 AI 엔진에 프롬프트를 전송하거나 무료 시뮬레이션 모드를 연산하여 의견을 취합하는 단계입니다.
3. **`가중치 합산 및 최종 결론 산출 완료.`**
   - 취합된 의견을 바탕으로 4영역 가중평균(AI 50%, 매크로 30%, 뉴스 15%, 소문 5%)을 계산해 최종 결론을 완성한 단계입니다.
4. **`분석 완료. [파일 > 보고서 저장]에서 보고서를 내보낼 수 있습니다.`**
   - 예측 대시보드 갱신이 완전히 끝나고 보고서 저장이 활성화된 최종 단계입니다.

## 5. GUI 화면 사용 설명서

### ① 상단: 최종 예측 요약 패널
- **최종 종합 예측 결과**: 4영역 가중 평균값을 기반으로 다음 날 **상승(▲, 초록색)** 혹은 **하락(▼, 밝은 연두색)** 여부, **예상 등락률(%)**, **예상 시초가(원)**를 큼직하게 시각화합니다. (분석 지표를 뜻하는 `ANALYTICS_ROUNDED` 아이콘이 적용되어 있습니다.)
- **가중치 배분 표시**: 상단 패널 우측에 통합 가중치 비율(`AI 50% │ 매크로 30% │ 뉴스 15% │ 소문 5%`)을 명시하여 계산 투명성을 높였습니다.
- **국내 기초자산 정보**: KODEX 200과 삼성전자, SK하이닉스의 현재가와 전일 대비 등락률을 보여줍니다. (높이가 120px로 30% 축소되어 콤팩트한 뷰를 제공합니다.)

### ② 최상단 헤더 및 컨트롤러
- **실시간 분석 기준 시각**: `YYYY년 MM월 DD일 HH시 MM분 기준` 텍스트 스팬이 **밝은 연두색 굵은 글씨**로 표시되어, 분석 실행 완료 시점의 현재 시각이 자동으로 갱신 기록됩니다.
- **[분석 실행]** 버튼: 최상단 타이틀 우측으로 이동되어 가로 너비가 126px로 30% 축소 및 굵은 글씨(`15pt`)로 변경되었으며, 클릭 시 실시간 분석을 가동합니다.
- **보고서 저장**: 화면 하단 버튼이 삭제되었으며, 상단 메뉴바의 **[파일] > [보고서 저장]**을 통해 결과를 내보납니다.
- **⚙️ 상단 [설정] 메뉴 > [API Key 설정]**: 각 AI 모델의 API Key를 등록하는 창이 나타납니다. 유료 전환 시 입력해 주시면 되며, 비워둘 시 무료 시뮬레이션 버전으로 구동됩니다.

### ③ 중간: 4대 AI 모델별 개별 예측 카드
