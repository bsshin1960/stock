# KODEX 200 AI 주식 예측 프로그램 사용 매뉴얼 (Plan & Help)

본 프로그램은 매 영업일 오후 3시 장 마감 시점의 국내외 금융 지표 및 시장 뉴스를 분석하여 다음 날 **KODEX 200 ETF의 시초가 등락을 예측**하는 인공지능 기반 주식 분석 솔루션입니다. 글로벌 4대 거대언어모델(LLM)인 Gemini, ChatGPT, Claude, Grok의 독립적인 분석 관점을 종합하는 집단 지성(Consensus) 아키텍처를 탑재하였습니다.

---

## 1. 주요 특징 및 핵심 기능
1. **정량/정성 데이터 통합 수집**:
   - KODEX 200 시세 데이터 및 기술적 지표(이동평균, RSI, MACD, 볼린저 밴드) 연산.
   - **삼성전자 및 SK하이닉스** 등 KODEX 200지수에 지대한 영향력을 가진 대형 기술주의 당일 주가 흐름 반영.
   - **미국 야간 선물(나스닥, S&P 500)** 및 **한국 선물(코스피 200 선물)**의 변동성 데이터 융합.
   - 원/달러 환율 및 **달러/엔 환율(일본 엔화 상황)**, **일본 닛케이 225(Nikkei 225)**의 변동성 추적. (수출 경쟁력 및 아시아 증시 동조성 분석)
   - 글로벌 시장의 심리를 보여주는 **VIX 공포 지수** 수집.
   - 당일 주요 포털 금융 뉴스 실시간 크롤링 및 신뢰성 있는 **증권가 소문/이슈(풍문)** 데이터를 반영.
2. **다중 AI 가중 종합 예측(Consensus)**:
   - 4대 AI(Gemini, ChatGPT, Claude, Grok) API에 분석 프롬프트 동시 전달.
   - **전문성 및 추론력 기반 가중치** 배분:
     - **Gemini**: `30%`
     - **ChatGPT**: `30%`
     - **Claude**: `25%`
     - **Grok**: `15%`
   - 각 AI가 제시한 등락률(%), 예상 시초가를 가중치 합산하여 소수점 둘째자리 수준의 최종 합의 결과 도출.
3. **사용자 친화적 Premium Dark UI & 한글 진행**:
   - 모던하고 미려한 다크 테마 기반 대시보드.
   - **데이터 수집부터 결과 도출까지 모든 진행 과정을 직관적인 한글 메시지로 표시.**
   - API Key를 GUI 내에서 로컬에 저장하고 안전하게 관리 (미입력 시 고품질 시뮬레이션 무료 구동).
4. **보고서 자동 저장**:
   - 매 예측 결과를 누적하여 Markdown 포맷 보고서(`reports/` 폴더 내 저장)로 기록.

## 2. 프로그램 구조 및 파일 구성
프로젝트는 모듈화된 파이썬 구조로 제작되어 유지보수와 기능 확장이 매우 용이합니다.

```
C:\Temp\Antigrvity\stock\
  ├─ main.py                # 프로그램 실행 메인 진입점
  ├─ requirements.txt       # 필요한 라이브러리 의존성 정의
  ├─ run.bat                # 윈도우 원클릭 자동 패키지 설치 및 실행 배치 파일
  ├─ .env                   # API Key 관리 환경 변수 설정 파일
  ├─ Plan & Help.md         # [본 문서] 사용자 가이드 및 설명서
  └─ src/
       ├─ __init__.py       # 패키지 선언 파일
       ├─ config.py         # 금융 지표 티커 설정 및 모델 목록 정의
       ├─ data_collector.py # 야후 파이낸스 API 연동 및 뉴스/소문 수집 모듈
       ├─ ai_consensus.py   # AI 모델 API 연동, 시뮬레이터, 가중평균 알고리즘
       ├─ reporter.py       # 예측 보고서 마크다운 변환 및 저장
       └─ app.py            # Flet GUI 컴포넌트 및 비동기 이벤트 핸들러
```

---

## 3. 프로그램 실행 방법 (Quick Start)

### 3.1. 원클릭 실행 (추천)
1. 프로그램 폴더 내에 있는 **`run.bat`** 파일을 더블 클릭하여 실행합니다.
2. 프로그램이 필요한 패키지(`flet`, `yfinance` 등)를 알아서 다운로드 및 설치한 후 예측 GUI 화면이 실행됩니다.

### 3.2. 수동 실행 (터미널)
파워쉘이나 cmd 터미널에서 다음 명령어를 순서대로 실행합니다:
```bash
# 1. 의존성 라이브러리 설치
pip install -r requirements.txt

# 2. 프로그램 가동
python main.py
```

## 4. 실시간 한글 진행 상황 (Process flow)
프로그램이 분석을 실행하면, GUI의 상태표시줄에 모든 분석 진척 상황이 **한글**로 알기 쉽게 표시됩니다:

1. **`대형주, 한일 선물지수, 환율 및 공포지수 크롤링 중...`**
   - KODEX 200, 삼성전자, SK하이닉스 가격 및 해외 선물 지수, 환율, VIX 인덱스 등의 데이터를 수집하는 단계입니다.
2. **`AI 모델 분석 시뮬레이션 및 API 응답 수립 중...`**
   - 4대 AI 엔진에 프롬프트를 전송하거나 무료 시뮬레이션 모드를 연산하여 의견을 취합하는 단계입니다.
3. **`가중치 합산 예측 및 최종 결론 산출 완료.`**
   - 취합된 의견을 Gemini(30%), ChatGPT(30%), Claude(25%), Grok(15%) 비율로 연산해 최종 결론을 완성한 단계입니다.
4. **`분석 완료. 리포트를 내보낼 수 있습니다.`**
   - 예측 대시보드 갱신이 완전히 끝나고 마크다운 파일 저장이 활성화된 최종 단계입니다.

## 5. GUI 화면 사용 설명서

### ① 상단: 최종 예측 요약 패널
- **최종 종합 예측 결과**: AI들의 가중 평균값을 기반으로 다음 날 **상승(▲) 혹은 하락(▼)** 여부, **예상 등락률(%)**, **예상 시초가(원)**를 큼직하게 시각화합니다.
- **가중치 배분 표시**: 상단 패널 우측에 모델별 가중치(Gemini 30%, ChatGPT 30% 등)를 명시해 계산 투명성을 높였습니다.
- **국내 기초자산 정보**: KODEX 200과 삼성전자, SK하이닉스의 현재가와 전일 대비 등락률을 보여줍니다.
- **⚙️ 우측 상단 톱니바퀴 버튼**: 각 AI 모델의 API Key를 등록하는 창이 나타납니다. 유료 전환 시 입력해 주시면 되며, 비워둘 시 무료 시뮬레이션 버전으로 구동됩니다.

### ② 중앙: 컨트롤바
- **[오후 3시 분석 실행]** 버튼: 클릭 시 한글 진행 과정 메시지와 함께 실시간 연산을 개시합니다.
- **[보고서 저장]** 버튼: 분석 완료 후 활성화되며, 현재 화면의 결과를 마크다운 파일로 `reports` 폴더에 내보냅니다.

### ③ 중간: 4대 AI 모델별 개별 예측 카드
- **Gemini (30%) / ChatGPT (30%) / Claude (25%) / Grok (15%)**의 카드입니다.
- 가중치가 이름 옆에 표기되며, 각 AI의 개별 예측률과 상승/하락 논리 근거(Reason)를 제공합니다.

### ④ 하단: 선물/환율/일본상황 실시간 대시보드
- 코스피 선물, 나스닥 선물, S&P 500 선물, 원/달러 환율, **엔/달러 환율**, **일본 닛케이**, VIX 공포지수, 국채금리, 유가 등 **총 9가지 거시경제 지표**를 카드 형태로 배치해 보여줍니다.

### ⑤ 최하단: 뉴스 및 소문 탭형 뷰
- **실시간 속보 뉴스** 탭과 **신뢰성 있는 증권가 소문/이슈** 탭을 제공하여, 투자 심리와 장외 찌라시 풍문 데이터를 비교해 볼 수 있습니다.
